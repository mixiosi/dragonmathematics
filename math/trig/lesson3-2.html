Mastering Triangle Solutions

Solve oblique triangles with precision using trigonometric relationships

The Law of Sines

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} \]

Key Applications

  • AAS/ASA Cases

    Solve triangles given two angles and one side

  • SSA Cases

    Handle ambiguous cases with careful analysis

Essential Formulas

\[ \text{Area} = \frac{1}{2}ab\sin C \]

Derived from basic trigonometric relationships

Worked Examples

Example 1: Find Missing Side

Given: ∠A = 30°, a = 10, ∠B = 45°

\[ \frac{10}{\sin 30°} = \frac{b}{\sin 45°} \Rightarrow b = \frac{10 \times \sin 45°}{\sin 30°} \approx 14.14 \]

Example 2: Ambiguous Case

Given: a = 7, b = 10, ∠A = 60°

\[ \sin B = \frac{10 \times \sin 60°}{7} \approx 1.237 \Rightarrow \text{No solution} \]

Practice Problems

Problem 1: Find Missing Angle

Given: ∠A = 40°, a = 8, b = 12

Using Law of Sines: \[ \frac{8}{\sin 40°} = \frac{12}{\sin B} \Rightarrow \sin B = \frac{12 \times \sin 40°}{8} \approx 0.964 \Rightarrow B \approx 74.6° \]

Ambiguous Case Conditions

  • Two solutions if: a < b and a > b sin A
  • One solution if: a ≥ b or a = b sin A
  • No solution if: a < b sin A
