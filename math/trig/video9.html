9. Trigonometric Identities

Key Identity Groups

Pythagorean

  • sin²θ + cos²θ = 1
  • 1 + tan²θ = sec²θ
  • 1 + cot²θ = csc²θ

Reciprocal

  • sinθ = 1/cscθ
  • cosθ = 1/secθ
  • tanθ = 1/cotθ

Quotient

  • tanθ = sinθ/cosθ
  • cotθ = cosθ/sinθ

Cofunction

  • sin(π/2 - θ) = cosθ
  • tan(π/2 - θ) = cotθ
  • sec(π/2 - θ) = cscθ

Simplification Strategies

  • Factor expressions where possible
  • Combine fractions using common denominators
  • Use Pythagorean identities to convert forms
  • Look for conjugate multiplications

Example Problem

Simplify: (1 - cos²θ)/(sinθ cosθ)

  1. Apply Pythagorean identity: 1 - cos²θ = sin²θ
  2. Rewrite expression: sin²θ/(sinθ cosθ)
  3. Simplify: sinθ/cosθ = tanθ

Learning Objectives

  • Recall fundamental trigonometric identities
  • Apply identities to simplify expressions
  • Develop identity recognition patterns
  • Solve complex trigonometric equations
  • Create proof strategies using identities
