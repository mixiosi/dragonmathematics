Double & Half Angle Formulas

Transform complex angles into manageable expressions

Key Angle Transformations

Double Angle Formulas

\[ \sin(2\theta) = 2\sin\theta\cos\theta \]
\[ \cos(2\theta) = \cos^2\theta - \sin^2\theta \]
\[ \tan(2\theta) = \frac{2\tan\theta}{1 - \tan^2\theta} \]

Half Angle Formulas

\[ \sin\left(\frac{\theta}{2}\right) = \pm\sqrt{\frac{1 - \cos\theta}{2}} \]
\[ \cos\left(\frac{\theta}{2}\right) = \pm\sqrt{\frac{1 + \cos\theta}{2}} \]
\[ \tan\left(\frac{\theta}{2}\right) = \frac{\sin\theta}{1 + \cos\theta} \]

Application Example

Find \(\sin(22.5^\circ)\) using half-angle formula

  1. Recognize angle relationship: \[ 22.5^\circ = \frac{45^\circ}{2} \]
  2. Apply half-angle formula: \[ \sin\left(\frac{45^\circ}{2}\right) = \sqrt{\frac{1 - \cos45^\circ}{2}} \]
  3. Calculate result: \[ = \sqrt{\frac{1 - \frac{\sqrt{2}}{2}}{2}} = \frac{\sqrt{2 - \sqrt{2}}}{2} \]

Practice Exercise

Prove: \(\cos(2\theta) = 1 - 2\sin^2\theta\)

Start with \(\cos(2\theta) = \cos^2\theta - \sin^2\theta\) and use Pythagorean identity:

\[ \cos^2\theta = 1 - \sin^2\theta \] \[ \cos(2\theta) = (1 - \sin^2\theta) - \sin^2\theta = 1 - 2\sin^2\theta \]
