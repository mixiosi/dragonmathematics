Vector Mathematics

Mastering Directional Quantities in Trigonometry

Core Vector Concepts

Vector Components

\[ \vec{v} = \|\vec{v}\|\cos\theta\,\hat{i} + \|\vec{v}\|\sin\theta\,\hat{j} \]

Where θ is angle from x-axis

Dot Product

\[ \vec{a} \cdot \vec{b} = \|\vec{a}\|\|\vec{b}\|\cos\theta \]

Measures vector alignment

Vector Operations

Addition
\[ \vec{a} + \vec{b} = (a_x + b_x)\hat{i} + (a_y + b_y)\hat{j} \]
Scalar Multiplication
\[ c\vec{a} = (ca_x)\hat{i} + (ca_y)\hat{j} \]
Magnitude
\[ \|\vec{a}\| = \sqrt{a_x^2 + a_y^2} \]

Worked Examples

Force Resolution

Given: 50N force at 30°

\[ F_x = 50\cos30° \approx 43.3\text{N} \]
\[ F_y = 50\sin30° = 25\text{N} \]

Angle Between Vectors

Given: \(\vec{a} = (3,4)\), \(\vec{b} = (4,3)\)

\[ \cos\theta = \frac{24}{5 \times 5} = 0.96 \Rightarrow \theta \approx 16.26° \]

Practice Exercise

For vectors \(\vec{u} = (2, -1)\) and \(\vec{v} = (3, 5)\):
  1. Calculate \(\vec{u} \cdot \vec{v}\)
  2. Find the angle between them

1. Dot product:

\[ 2(3) + (-1)(5) = 6 - 5 = 1 \]

2. Angle calculation:

\[ \|\vec{u}\| = \sqrt{5},\ \|\vec{v}\| = \sqrt{34} \] \[ \theta = \arccos\left(\frac{1}{\sqrt{5 \times 34}}\right) \approx 85.14° \]
