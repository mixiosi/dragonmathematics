6. Graphing Sine & Cosine Functions

Key Concepts

  • Standard form: y = A sin(B(x - C)) + D
  • Amplitude, period, and frequency relationships
  • Phase shift and vertical shift effects
  • Identifying critical points on waveforms
  • Transformation parameter analysis

Amplitude

|A| = Vertical stretch/compression
Determines wave height

Period

2π/|B| = Horizontal stretch/compression
Controls wave length

Transformation Guide

  • A: Vertical scaling (amplitude)
  • B: Horizontal scaling (period)
  • C: Horizontal shift (phase shift)
  • D: Vertical shift

Example: Graph y = 2 sin(3x - π/4) + 1

  1. Amplitude = 2
  2. Period = 2π/3
  3. Phase shift = π/12 right
  4. Vertical shift = 1 up
  5. Plot key points and connect

Learning Objectives

  • Sketch transformed sine/cosine graphs
  • Analyze parameters in function equations
  • Determine domain/range of transformed functions
  • Connect algebraic form to graphical features
  • Solve real-world periodic phenomena problems
