Law of Cosines

Extending the Pythagorean Theorem to All Triangles

Fundamental Formula

\[ c^2 = a^2 + b^2 - 2ab\cos(C) \]

Key Applications

Case 1: SAS (Side-Angle-Side)

Given two sides and the included angle:

\[ c = \sqrt{a^2 + b^2 - 2ab\cos(C)} \]

Case 2: SSS (Side-Side-Side)

Given three sides, find any angle:

\[ \cos(C) = \frac{a^2 + b^2 - c^2}{2ab} \]

Step-by-Step Problem Solving

Example 1: Find Missing Side (SAS)

Given: a = 8, b = 6, ∠C = 60°

Solution:

\[ c^2 = 8^2 + 6^2 - 2(8)(6)\cos(60°) \] \[ c^2 = 64 + 36 - 96(0.5) = 52 \] \[ c = \sqrt{52} \approx 7.21 \]

Example 2: Find Missing Angle (SSS)

Given: a = 7, b = 10, c = 5

Solution:

\[ \cos(C) = \frac{7^2 + 10^2 - 5^2}{2(7)(10)} = \frac{124}{140} \approx 0.8857 \] \[ C = \arccos(0.8857) \approx 27.5° \]

Comparison with Law of Sines

Law of Cosines Law of Sines
Works for SAS and SSS cases Works for AAS and SSA cases
Requires one angle for SSS Requires two angles for SSA
Always unique solution Possible ambiguous case

Practice Exercise

Triangle ABC has sides a = 9, b = 12, c = 15. Find ∠C.
\[ \cos(C) = \frac{9^2 + 12^2 - 15^2}{2(9)(12)} = \frac{81 + 144 - 225}{216} = 0 \] \[ C = \arccos(0) = 90° \]
