Product-to-Sum & Sum-to-Product Formulas

Master trigonometric transformations for advanced problem solving

Essential Transformation Formulas

Product-to-Sum Identities

\(\sin A \sin B = \frac{1}{2}[\cos(A-B) - \cos(A+B)]\)
\(\cos A \cos B = \frac{1}{2}[\cos(A-B) + \cos(A+B)]\)
\(\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]\)
\(\cos A \sin B = \frac{1}{2}[\sin(A+B) - \sin(A-B)]\)

Sum-to-Product Identities

\(\sin A + \sin B = 2\sin\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)\)
\(\sin A - \sin B = 2\cos\left(\frac{A+B}{2}\right)\sin\left(\frac{A-B}{2}\right)\)
\(\cos A + \cos B = 2\cos\left(\frac{A+B}{2}\right)\cos\left(\frac{A-B}{2}\right)\)
\(\cos A - \cos B = -2\sin\left(\frac{A+B}{2}\right)\sin\left(\frac{A-B}{2}\right)\)

Example 1: Product-to-Sum Conversion

Convert \( \sin 3x \cos 5x \) using product-to-sum formula:

\[ \sin 3x \cos 5x = \frac{1}{2}[\sin(8x) + \sin(-2x)] = \frac{1}{2}(\sin 8x - \sin 2x) \]

Example 2: Sum-to-Product Conversion

Simplify \( \cos 75° - \cos 15° \):

\[ \cos 75° - \cos 15° = -2\sin\left(\frac{90°}{2}\right)\sin\left(\frac{60°}{2}\right) = -2\sin 45°\sin 30° = -\sqrt{2} \]

Applications

  • Simplifying complex trigonometric expressions
  • Solving trigonometric equations
  • Calculating integrals in calculus
  • Analyzing wave interference patterns
